# Contributing to Pin-Board

We love your input! We want to make contributing to Pin-Board as easy and transparent as possible.

## How to Contribute
1.  **Fork the repo** and create your branch from `master`.
2.  **Clone** your fork.
3.  **Make your changes**.
4.  **Test** your changes using `cargo run`.
5.  **Submit a Pull Request**.

## Reporting Issues
We use GitHub issues to track public bugs. Report a bug by opening a new issue; it's that easy!
